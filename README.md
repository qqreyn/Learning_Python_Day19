# Learning_Python_Day19

Today's topic is working with APIs and JSON data!

**Goal:** Parse and manipulate JSON data from real APIs

- **task1:** Fetch JSON data from a weather API (for example OpenWeatherMap)

- **task2:** Parse the API response into Python dictionaries

- **task3:** Access nested values (e.g. `response['key']['subkey']`)

- **task4:** Iterate through JSON arrays

- **task5:** Filter data based on conditions

- **task6:** Use `json.dumps()` to format JSON output

**mini-project:**
Weather Data Analyzer:

- Connect to a free weather API  
- Ask the user for a city name  
- Fetch current weather data  
- Parse the JSON response  
- Extract important information: temperature, weather conditions, humidity  
- Display the results in a clear formatted output  
- Handle errors such as invalid city names or failed requests
